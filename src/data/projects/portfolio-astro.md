# Portfolio Personale con Astro

Un portfolio personale di altissima qualità, costruito con le tecnologie più moderne.

## Caratteristiche

- **Design Stile Apple**: Ampi spazi bianchi, tipografia pulita e glassmorphism
- **Animazioni GSAP**: Animazioni fluide e performanti con ScrollTrigger
- **Modello 3D**: Dente stilizzato in Three.js con mouse tracking
- **Dark Mode**: Supporto completo per tema scuro
- **CMS Integrato**: Decap CMS per gestire contenuti senza codice
- **Performance**: Lighthouse score > 90 su desktop

## Tecnologie Utilizzate

- Astro 5
- Tailwind CSS v4
- GSAP (ScrollTrigger, SplitText)
- Three.js
- React
- Decap CMS

## Risultati

- Tempo di caricamento: < 2 secondi
- Lighthouse Score: 95+
- Mobile Responsive: 100%
